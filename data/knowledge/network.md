# 网络与 IT 故障处理指南

## WiFi 连不上怎么办？
- 请首先确保您选择了正确的内部无线网络名称：`Corp_Staff` 或 `Corp_Guest`。
- 员工请使用统一的 OA 账号及密码认证。访客需由前台分配临时邀请码。
- 如果一直提示“认证失败”，请重启电脑，或者在网络设置中将 DNS 手动修改为 `114.114.114.114`。

## VPN 无法访问内网
- 请检查您的 VPN 客户端版本是否最新，旧版本可能会因为证书过期导致无法连接。
- 尝试切换连接节点，优先选择延迟低于 50ms 的节点。

## 打印机无法打印 / 打印机连不上
- 请确认您是否已经连接到了公司内网 `Corp_Staff`。
- 如果已经连接内网，请在“设备与打印机”中重新添加共享打印机，IP 为 `192.168.1.100`。
- 若提示“缺纸”或“卡纸”，请联系行政部门处理，不要强行拉扯纸张。
